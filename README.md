## Movie Recommendation System

A Python Movie Recommendation System built using loops and intermediate Python knowledge.

# Why I Built This
I wanted to create a simple recommendation system that helps users discover movies based on their preferences. This project helped me practice organizing data with Python dictionaries and designing interactive command-line applications.

# Features

- Can suggest some fan favourite Movies
- Suggest movies in short and long form

# Technologies Used

- Python

# What I Learned

While building this project I learned:

- Handling Loops
- Working with lists
- User input handling
